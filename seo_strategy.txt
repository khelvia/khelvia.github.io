## Performance Strategy (Pending)

### 1. Lazy Loading
- [ ] Implement `loading="lazy"` on all images below the fold.
- [ ] Use `loading="eager"` for the LCP (Largest Contentful Paint) image (hero image).

- [ ] Explicitly set `width` and `height` attributes on `<img>` tags to prevent Cumulative Layout Shift (CLS).

- [ ] Convert images to WebP format where possible.
- [ ] Ensure existing PNG/JPGs are optimized.

### 4. Resource Hints
- [x] Use `<link rel="preconnect">` for Google Fonts.
- [ ] Use `decoding="async"` for non-critical images.
